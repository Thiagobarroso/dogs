
# Dogs - Rede Social para Cachorros

**Dogs** é uma rede social feita para cachorros, onde seus donos podem compartilhar fotos, histórias e interagir com outros donos de dogs. Este projeto está sendo desenvolvido com **React**, **Vite** e **React Router**.

## Tecnologias Usadas

- **React**: Para criar a interface do usuário.
- **Vite**: Para build e desenvolvimento rápido.
- **React Router**: Para navegação entre páginas.

## Funcionalidades

- Publicação de fotos e posts.
- Perfil do cachorro com informações e fotos.
- Seguir outros cachorros e interagir nos posts.

## Rodando o Projeto

1. Clone o repositório:
   ```bash
   git clone https://github.com/thiagobarroso/dogs-rede-social.git
   ```
   
2. Instale as dependências:
   ```bash
   cd dogs-rede-social
   npm install
   ```
   
3. Inicie o projeto:
   ```bash
   npm run dev
   ```

O projeto estará rodando em `http://localhost:5173`.
